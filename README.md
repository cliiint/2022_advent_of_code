# 2022 Advent of Code
This is my attempt at the [2022 Advent of Code](https://adventofcode.com/) in Ruby.

To execute a day, run the rake task and pass in the number of the day as an argument.
```bash
rake "run[3]"
```

To run unit tests
```bash
rspec
```
